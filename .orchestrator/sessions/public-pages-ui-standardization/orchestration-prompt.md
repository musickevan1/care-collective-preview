## Phase 0: Planning & Design Decisions

### 0.1 Answer Design Questions

Before starting, answer these questions:
1. **Icon Backgrounds**: Use solid tints (`bg-sage/10`, `bg-primary/10`) or keep gradients (`from-sage to-sage-dark`)?
   - **Recommendation**: Solid tints - cleaner, more modern, matches existing help page
2. **Card Borders**: Restrict to 2-3 variants or allow custom colors?
   - **Recommendation**: 4 strict variants (standard, featured, accent, warning) - ensures consistency
3. **Help Page Colors**: Keep `text-secondary` (dusty rose) or change to `text-foreground`?
   - **Recommendation**: Change to `text-foreground` - consistency is more important than page distinction
4. **Privacy & Terms Headers**: Add header icons or stay minimal?
   - **Recommendation**: Add header icons - visual consistency and professional appearance
5. **Implementation Scope**: All improvements at once (7-10 hours) or break into multiple sessions?
   - **Recommendation**: Complete in one session - high impact improvements deserve full effort

```typescript

    <CardTitle>Card Title</CardTitle>
  </CardHeader>

    Card content
  </CardContent>
</Card>

// Variant 3: Accent Card (highlights/special features)

  <CardContent className="p-6">
    Card content
  </CardContent>
</Card>

// Variant 4: Warning Card (alerts/important notices)
<Card className="border-dusty-rose/30 bg-dusty-rose/5">
  <CardContent className="p-6">
    Card content
  </CardContent>
</Card>
```

**Apply to:** All cards across all pages consistently

## Design System Reference

### Color Semantics
```typescript
const colorSemantics = {
  // Primary Actions/Branding - CTAs, active states, primary links
  sage: '#7A9E99',

  // Important/Featured Content - Section headers, featured cards, important info
  primary: '#5A7D78',

  // Standard Content - Body text, page titles, card titles
  foreground: '#483129',

  // Highlights/Features - Special features, decorative elements
  accent: '#C39778',

  // Warnings/Important Notices - Alerts, warnings, important notices
  dustyRose: '#D8A8A0',
}
```

### Component Patterns

**Page Header:**
```tsx
<PageHeader
  title="Page Title"
  description="Optional description"
  icon={<Icon />}
  iconBgColor="sage" | "primary" | "dusty-rose" | "accent"
/>
```

**Section Header:**
```tsx
<SectionHeader
  title="Section Title"
  description="Optional description"
  icon={<Icon />}
  iconBgColor="primary" | "sage" | "dusty-rose"
/>
```

**Cards:**
```tsx
// Standard Card
<Card className="bg-white border-sage/20 shadow-md hover:shadow-lg transition-all duration-300">
  <CardHeader><CardTitle>Title</CardTitle></CardHeader>
  <CardContent>Content</CardContent>
</Card>

// Featured Card
<Card className="bg-white border-sage/30 shadow-lg hover:shadow-xl hover:border-sage hover:-translate-y-1">
  <CardHeader><CardTitle>Title</CardTitle></CardHeader>
  <CardContent>Content</CardContent>
</Card>

// Accent Card
<Card className="border-primary/20 bg-primary/5">
  <CardContent>Content</CardContent>
</Card>

  <CardContent>Content</CardContent>
</Card>
```
