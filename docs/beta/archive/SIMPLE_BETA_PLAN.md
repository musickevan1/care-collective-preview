# Care Collective Beta Test - Simple Plan

## Overview

**What**: Testing the Care Collective platform with 5 friends/family
**How Long**: 1-2 weeks of casual testing
**Goal**: Make sure everything works and gather feedback

---

## What We're Testing

### Core Features
1. **Help Requests** - Creating and browsing requests for help
2. **Messaging** - Connecting helpers and requesters
3. **Mobile Experience** - Making sure it works on phones
4. **Overall Usability** - Is it easy to use?

---

## What Testers Need to Do

### Week 1: Basic Testing (20-30 minutes)

**Day 1-2: Get Started**
1. Log in with the credentials we'll send you
2. Look around and explore the platform
3. Create at least one help request (any category)
4. Browse other people's requests

**Day 3-7: Try the Main Features**
1. Offer to help on someone's request
2. Send a few messages back and forth
3. Try it on your phone if you can
4. Update a request status if you created one

### Week 2: Final Testing (10-15 minutes)

**Day 8-14: Wrap Up**
1. Try anything you haven't tested yet
2. Create another request or respond to one
3. Fill out a quick feedback form we'll send
4. Report any bugs or confusing things you found

## Quick Scenarios to Try

### Scenario 1: "I Need Help"
1. Log in
2. Click "Create Request"
3. Fill out the form (groceries, transport, etc.)
4. Submit and see it appear in the browse page

### Scenario 2: "I Can Help"
1. Browse available requests
2. Click on one that interests you
3. Click "Offer Help"
4. Send a message to the person

### Scenario 3: "Mobile Test"
1. Open the site on your phone
2. Try creating a request
3. Try responding to a request
4. Check if messaging works

### Scenario 4: "Full Journey"
1. Person A creates a help request
2. Person B offers to help
3. They message back and forth
4. Person A marks the request as complete
