## Filtering Tips

### On Unsplash
1. Use search bar at top
2. Filter by "Landscape" orientation for hero images
3. Click on image ‚Üí Check photographer allows commercial use (Unsplash license = always free)
4. Download highest resolution

### On Pexels
1. Use search bar
2. Filter: "Orientation: Horizontal"
3. Filter: "Size: Large" (for high resolution)
4. Green "Free Download" button = safe to use
5. All Pexels images are free for commercial use

### On Pixabay
1. Use search bar
2. Filter: "Horizontal" orientation
3. Check for "Pixabay License" badge
4. Download "Original size" for best quality

## Pro Tips

### For Better Results:
1. **Use multiple search terms** - Don't rely on one search
2. **Check related images** - When you find a good one, click on similar tags
3. **Browse by photographer** - If you find a good image, check that photographer's other work
4. **Try "community" searches** - Community-related images often have the peer dynamic you want
5. **Filter by date** - Newer uploads often have better quality/diversity

### Text Overlay Considerations:
- Choose images with clear space for text (top or center)
- Avoid busy backgrounds that compete with text
- Test with dark overlay (most hero text is white)
- Ensure faces are visible and in focus
