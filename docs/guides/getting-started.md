# Getting Started with Care Collective

This guide helps new developers set up their development environment and make their first contribution.

## Prerequisites

- Node.js 18+
- npm or pnpm
- Git
- Supabase CLI (optional, for local database)

## Quick Start

```bash
# Clone the repository
git clone https://github.com/your-org/care-collective.git
cd care-collective

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Edit .env.local with your Supabase credentials

# Start development server
npm run dev
```
