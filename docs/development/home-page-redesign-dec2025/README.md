### What's Changing:
1. **Font sizes** - Increased across all text elements
2. **Color scheme** - Teal (#3D4F52) + rose for home page accents
3. **Section backgrounds** - Color-blocked with logo colors
4. **Content structure** - New "What is CARE Collective?" 3-box section
5. **About section** - Maureen's photo, asymmetrical layout
6. **Footer** - MSU/SGS funding statement, Facebook icon
