### Styling Scope

#### 1. Gradient Definitions

Create a cohesive gradient system in `app/globals.css`:

```css
/* Section gradient presets */
:root {
  /* Hero gradient - warm, welcoming */
  --gradient-hero: linear-gradient(
    135deg,
    var(--color-background) 0%,
    color-mix(in srgb, var(--color-sage-light) 10%, var(--color-background)) 50%,
    color-mix(in srgb, var(--color-dusty-rose-light) 5%, var(--color-background)) 100%
  );
  
  /* Section accent gradients */
  --gradient-sage-subtle: linear-gradient(
    180deg,
    color-mix(in srgb, var(--color-sage) 8%, transparent) 0%,
    color-mix(in srgb, var(--color-sage) 15%, transparent) 100%
  );
  
  --gradient-dusty-rose-subtle: linear-gradient(
    180deg,
    color-mix(in srgb, var(--color-dusty-rose) 10%, transparent) 0%,
    color-mix(in srgb, var(--color-dusty-rose) 20%, transparent) 100%
  );
  
  /* Dark section gradients */
  --gradient-navy-rich: linear-gradient(
    180deg,
    var(--color-navy) 0%,
    color-mix(in srgb, var(--color-navy) 90%, black) 100%
  );
  
  --gradient-sage-dark-rich: linear-gradient(
    180deg,
    var(--color-sage-dark) 0%,
    color-mix(in srgb, var(--color-sage-dark) 85%, black) 100%
  );
}
```

**Gradient Requirements:**
- Use `color-mix()` for modern, precise color blending
- Fallbacks for older browsers
- Consistent angle system (180deg for vertical, 135deg for diagonal)
- Subtle variations, not dramatic shifts

#### 2. Animation & Transition Effects

**Section Reveal Animations:**
```css
/* Scroll-triggered fade-in for sections */
@keyframes section-reveal {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

.section-animate {
  animation: section-reveal 0.6s ease-out forwards;
  animation-play-state: paused;
}

.section-animate.is-visible {
  animation-play-state: running;
}
```

**Divider Animations:**
```css
/* Subtle wave motion for dividers (optional, reduced-motion aware) */
@keyframes wave-subtle {
  0%, 100% { transform: translateX(0); }
  50% { transform: translateX(-5px); }
}

.divider-animated {
  animation: wave-subtle 8s ease-in-out infinite;
}

@media (prefers-reduced-motion: reduce) {
  .divider-animated,
  .section-animate {
    animation: none;
    opacity: 1;
    transform: none;
  }
}
```

**Hover & Interaction Effects:**
```css
/* Card hover on colored backgrounds */
.card-elevated {
  transition: transform 0.3s ease, box-shadow 0.3s ease;
  box-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.1);
}

.card-elevated:hover {
  transform: translateY(-4px);
  box-shadow: 0 20px 25px -5px rgba(0, 0, 0, 0.1),
              0 10px 10px -5px rgba(0, 0, 0, 0.04);
}

/* On dark backgrounds, adjust shadow color */
.bg-dark .card-elevated:hover {
  box-shadow: 0 20px 25px -5px rgba(0, 0, 0, 0.3),
              0 10px 10px -5px rgba(0, 0, 0, 0.2);
}
```

### Quality Checklist

**Visual Consistency:**
- [ ] Gradient angles consistent across sections
- [ ] Divider curves feel organic, not mathematical
- [ ] Card shadows consistent regardless of background
- [ ] Animation timing feels natural (0.2-0.4s for micro, 0.6-1s for section)

**Performance:**
- [ ] No layout shift during animations
- [ ] GPU-accelerated transforms (`transform`, `opacity` only)
- [ ] No expensive `filter` properties on scroll
- [ ] Lighthouse Performance score > 90

**Accessibility:**
- [ ] `prefers-reduced-motion` respected everywhere
- [ ] Focus indicators visible on all backgrounds
- [ ] Text remains readable at 200% zoom
- [ ] Color not sole means of conveying information

**Browser Testing:**
- [ ] Chrome 120+ (primary)
- [ ] Firefox 120+
- [ ] Safari 17+
- [ ] Edge 120+
- [ ] iOS Safari 17+
- [ ] Chrome Android

### Success Criteria

- [ ] Transitions between sections feel seamless
- [ ] No visible color banding in gradients
- [ ] Animations enhance rather than distract
- [ ] Mobile experience feels polished, not compromised
- [ ] Dark mode ready (if applicable)
